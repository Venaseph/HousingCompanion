HOUSING COMPANION v2.1.0
===============================================

WHAT'S INCLUDED IN THIS ZIP:
- The complete Housing Companion addon
- HousingCompanion folder - Extract to Interface/AddOns/
- BUGFIX_SUMMARY.txt - Latest fixes and changes

INSTALLATION:
1. Extract HousingCompanion folder to: 
   Interface/AddOns/ (in your WoW directory)
2. Restart World of Warcraft
3. Type /hc to open

SLASH COMMANDS:
/hc or /housingcompanion - Toggle window open/closed

FEATURES:
✓ Endeavors - Track neighborhood tasks
✓ Rewards - Achievements and quest tracking  
✓ Vendors - Browse housing decor vendors
✓ Reputations - Track reputation standing (NEW!)
✓ Leaderboard - Neighborhood rankings
✓ Activity - Community completion feed
✓ Info - How-to guide
✓ Config - Settings and customization
